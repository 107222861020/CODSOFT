SMART TO-DO MANAGER PROJECT


Project Description:
A Python based desktop application used to manage daily tasks.


Features:
- User Login System
- Create Account
- Add Tasks
- Delete Tasks
- Mark Tasks Completed
- Priority Management
- Search Tasks
- SQLite Database Storage


Technologies Used:

Python
Tkinter GUI
SQLite Database

Conclusion:

This project helps users organize and track their daily tasks efficiently.
